**Defense Management System (MySQL)**

This project is a database management system developed using MySQL Command Line Client. The system is designed to manage and organize defense-related information such as personnel details, military units, operations, training records, and assets.

**Project Overview**
The Defense Management System stores and manages data related to military operations and resources. It demonstrates the use of SQL queries and database design concepts such as table creation, data insertion, and data retrieval.

**Database Tables**

1. Personnel
   This table stores information about military personnel such as ID, name, rank, and assigned unit.

2. MilitaryUnits
   This table contains information about different military units or divisions.

3. Operations
   This table records details about defense operations or missions.

4. Training
   This table keeps track of training programs and activities conducted for personnel.

5. Assets
   This table manages information about military assets such as equipment, vehicles, or weapons.

**Technologies Used**
MySQL
SQL (Structured Query Language)

**Features**
Database schema design using SQL
Multiple related tables for structured data management
Data insertion and retrieval using SQL queries
Command Line based MySQL database implementation

**How to Use the Project**

1. Install MySQL on your system.
2. Open MySQL Command Line Client.
3. Import the SQL file using the command:

SOURCE defense.sql;

4. The database and tables will be created automatically.
